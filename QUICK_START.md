# Hướng Dẫn Nhanh

## Chạy Ứng Dụng

### Cách 1: Chạy từ Python (Development)
```bash
python main.py
```

### Cách 2: Chạy file EXE (Production)
Double-click vào file: `dist\QuanLyQuanNhan.exe`

## Build Lại Ứng Dụng

### Sử dụng PowerShell (Khuyên dùng)
```powershell
.\build.ps1
```

### Sử dụng Python
```bash
python build_exe.py
```

File EXE sẽ được tạo tại: `dist\QuanLyQuanNhan.exe`
